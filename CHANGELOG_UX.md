## 📦 ما تغير

### ✨ ميزة 1: نظام Popups أنيق بدل الحوارات البيضاء

**المشكلة السابقة:**
- كل `alert()` كان بيظهر popup أبيض قبيح من المتصفح
- كل `confirm()` نفس الحاجة بزر OK/Cancel بالإنجليزي
- كل `prompt()` بيظهر input box من المتصفح مش مناسب للعربي

**التحسين:**
- **24 استخدام** اتبدلوا بـ popups أنيقة بنفس شكل البرنامج
- Cairo font عربي، RTL، ألوان متسقة
- 4 أنواع من الـ popups:
  - `ask()` — تأكيد نعم/لا مع خيار "danger" للعمليات الخطيرة
  - `tell()` — رسالة معلوماتية (نجاح/تحذير/خطأ)
  - `askInput()` — إدخال قيمة واحدة مع validation
  - `askForm()` — إدخال متعدد الحقول في popup واحد

**مميزات تفاعلية:**
- ⌨️ **Escape** يقفل
- ⌨️ **Enter** يأكد/ينتقل للحقل التالي
- 🖱️ **Click outside** يقفل (للـ ask/askInput/askForm فقط)
- ✅ **Validation inline** تحت كل حقل (مش popup تاني)
- 🎯 **Auto-focus** على أول input عند الفتح

**مثال — بيع كسر (قبل):**
```
Popup 1: "بيع كسر — ادخل رقم الموديل:" → يدخل
Popup 2: "الكمية:" → يدخل
(لو الموديل غلط، الرسالة بتظهر بعد ما كمل!)
```

**بيع كسر (بعد):**
```
┌──────────────────────────┐
│  بيع كسر                │
│  رقم الموديل:            │
│  [_______________]       │
│  الكمية:                 │
│  [_______________]       │
│  [إلغاء]    [تأكيد]      │
└──────────────────────────┘
(Validation مباشر — الموديل الغلط يظهر خطأ قبل ما يكمل)
```

### ⚡ ميزة 2: Debouncing في البحث — لا مزيد من اللاج

**المشكلة السابقة:**
- لما كنت تكتب في البحث السريع، كل حرف بيعمل filter على كل الأوردرات (500+)
- النتيجة: lag واضح على الموبايل، البرنامج يحس إنه بيعلق

**التحسين:**
- تم إضافة `useDebounced` hook
- الـ input يتحدث **فوراً** (سلس زي WhatsApp)
- الـ **filter** يستنى **250ms** بعد آخر حرف ثم ينفذ
- النتيجة: تجربة كتابة سلسة، بدون lag

**الـ searches المتأثرة:**
1. `gSearch` — البحث السريع في الهيدر (250ms)
2. `rcvSearch` — بحث الاستلام من الورش (250ms)
3. `filterSearch` — فلتر الخزنة (250ms)
4. `partySearch` — بحث العملاء/الموردين (200ms)

- `gSearch` → `gSearchDeb` في البحث السريع
- `rcvSearch` → `rcvSearchDeb` في بحث الورش
- `filterSearch` → `filterSearchDeb` في فلتر الخزنة
- `partySearch` → `partySearchDeb` في بحث العملاء (موقعين)
