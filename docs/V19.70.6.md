## 1. Recurring campaigns

### Recurrence types

| Type | Inputs | Behavior |
|---|---|---|
| `once` (default) | datetime | Fires once at the chosen datetime, then status="done" |
| `daily` | timeOfDay | Fires every day at timeOfDay (Cairo) |
| `weekly` | daysOfWeek[] + timeOfDay | Fires on selected days each week (e.g. Mon, Wed) |
| `monthly` | dayOfMonth (1-28) + timeOfDay | Fires once a month on the chosen day. Capped at 28 to avoid Feb edge cases. |
| `range` | rangeStart..rangeEnd + timeOfDay | Daily fire within the date range, then stops |

### End-conditions

For `daily`/`weekly`/`monthly` (which would otherwise fire forever):
- `maxOccurrences` — stop after N successful fires
- `endDate` — stop after this date (inclusive)
- Whichever fires first

For `range`: implicit end at `rangeEnd`.

### Data model

```js
data.scheduledCampaigns = [
  {
    id, templateId, templateName, templateBody,
    segmentKey, segmentLabel, items[], images[],
    scheduledAt,/* NEXT fire ISO — updated after each fire for recurring */
    status: "scheduled" | "firing" | "done" | "failed" | "cancelled",
    recurrence: null | {
      type: "daily"|"weekly"|"monthly"|"range",
      timeOfDay: "HH:MM",
      daysOfWeek: [],/* for weekly */
      dayOfMonth: null,/* for monthly */
      rangeStart, rangeEnd,/* for range */
      maxOccurrences: null,
      endDate: null,
    },
    occurrenceCount: 0,/* incremented after each successful fire */
    lastFiredAt: null,/* ISO of most recent fire */
    sentCount: 0,/* total messages sent across all occurrences */
    // ...
  }
]
```

### Cron flow

```
Tick (every 5 min)
  ↓
For each `scheduled` entry where scheduledAt <= now:
  ↓
  Transactional claim (status="firing")
  ↓
  Build messages + media
  ↓
  POST bridge /send
  ↓
  On success:
    occurrenceCount++
    lastFiredAt = now
    sentCount += accepted
    if recurrence:
      nextIso = computeNextFireTime(recurrence, lastFiredAt, occurrenceCount)
      if nextIso != null:
        status = "scheduled", scheduledAt = nextIso
      else:
        status = "done"
    else:
      status = "done"
```

### Sample patterns

**Daily morning thank-you:**
```
type: "daily", timeOfDay: "09:00"
```
Fires every day at 9 AM forever (until cancelled).

**Weekly promo (Mon + Wed):**
```
type: "weekly", daysOfWeek: [1, 3], timeOfDay: "10:00", maxOccurrences: 12
```
Fires Mon + Wed at 10 AM, stops after 12 fires (= 6 weeks).

**Monthly invoice reminder:**
```
type: "monthly", dayOfMonth: 28, timeOfDay: "11:00", endDate: "2026-12-31"
```
Fires on the 28th of each month at 11 AM until end of 2026.

**Date-range campaign:**
```
type: "range", rangeStart: "2026-05-15", rangeEnd: "2026-05-30", timeOfDay: "14:00"
```
Daily fire from May 15 to May 30 at 2 PM.
