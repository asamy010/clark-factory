## ✅ ما تم إنجازه

### 1. 🖨️ طباعة تقرير الحركات (Movements Report)

تقرير احترافي يشمل:

**Header:**
- 📊 عنوان "تقرير حركات المخزن"
- تاريخ الطباعة
- إجمالي عدد الحركات

**Summary Cards (4 خانات):**
- خامات: عدد + دخول + خروج
- إكسسوار: عدد + دخول + خروج
- منتجات عامة: عدد + دخول + خروج
- **قيمة الدخول الإجمالية** (بالجنيه)

**تفاصيل الفلاتر المطبقة:**
```
الفلاتر المطبقة: من: 2026-01-01 • إلى: 2026-04-19 • النوع: دخول
```

**جدول شامل** مع:
- التاريخ، النوع، الحركة
- الصنف، الكمية، السعر
- الملاحظات، بواسطة

**تحسينات:**
- ألوان حسب النوع (أحمر لخروج، أخضر لدخول)
- Page break تلقائي في الطباعة
- Print trigger تلقائي

### 2. 🖨️ طباعة جرد المخزن الشامل

من **نظرة عامة**، زر "🖨️ جرد شامل":

**محتوى التقرير:**
- الأرصدة الحالية لكل صنف (خامات + إكسسوار + منتجات)
- النوع + الفئة + الاسم + الرصيد + الحد الأدنى + التكلفة + القيمة + الحالة
- صف إجمالي كلي لقيمة المخزن
- **3 خانات توقيع:** مسؤول المخزن / المحاسب / المدير

### 3. 📊 Export CSV

#### Export سجل الحركات:
- زر "📊 Excel" في sub-tab الحركات
- يحمل الحركات المفلترة فقط
- اسم الملف: `stock-movements-2026-04-19.csv`

#### Export جرد المخزن:
- زر "📊 Excel" في نظرة عامة
- كل الأصناف الحالية (خامات + إكسسوار + منتجات) مع أرصدتهم
- اسم الملف: `stock-snapshot-2026-04-19.csv`

**مميزات فنية:**
- ✅ BOM للدعم الكامل للعربية في Excel
- ✅ Escape للـ quotes و commas
- ✅ Encoding UTF-8
- ✅ يفتح مباشرة في Excel / Google Sheets

### 4. 📱 QR Code للمنتجات العامة

زر جديد **📱** بجانب كل منتج عام:

**محتوى الـ QR:**
```json
{
  "app": "clark",
  "type": "prod",
  "id": "...",
  "name": "..."
}
```

**صفحة الطباعة:**
- Card 60mm × auto
- QR size: 45mm
- اسم المنتج + الوحدة + الرصيد الحالي
- Badge بالفئة
- Auto-print عند الفتح

**الفائدة:**
- يمكن لصق QR على المنتج في المخزن
- Scan في المستقبل للوصول السريع (بيتم في جلسة C)

### 5. 🚨 Dashboard Alerts — المنتجات العامة

قبل: التنبيهات للخامات والإكسسوار فقط  
بعد: **كمان المنتجات العامة** 🎊

```
🚫 مستلزمات تشغيل — زيت ماكينات SAE 30 نفذ من المخزن!
⚠️ قطع غيار — مواسير تزييت ناقص (2 قطعة — الحد الأدنى 5)
```

**فرق مهم:** تنبيهات المنتجات العامة **لا تعتمد على `stockEnabled`** — شغالة دائماً (لأن المنتجات العامة مش مرتبطة بالأوردرات).

## 💡 تصميم الـ CSV

### مزايا Implementation:

```javascript
const BOM = "\uFEFF";  // Excel يفهم UTF-8 بالبون
const escape = (v) => {
  if (v == null) return "";
  const s = String(v);
  if (/[",\n]/.test(s)) return '"' + s.replace(/"/g, '""') + '"';
  return s;
};
```

**ليه BOM؟**
- بدون BOM: Excel يعرض العربية بطريقة غلط
- مع BOM: يفتح الملف ويعرض العربي صح

**ليه Escape؟**
- لو قيمة فيها comma: لازم تتحاط بـ `"..."`
- لو فيها `"`: لازم تتحول لـ `""`
