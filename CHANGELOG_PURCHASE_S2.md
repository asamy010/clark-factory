### 3. Weighted Average Cost ✨

**الـ logic:**
```javascript
newAvgCost = (oldStock × oldAvgCost + newQty × newPrice) / (oldStock + newQty)
```

**مثال:**
- الرصيد: 10 متر بسعر 100 ج
- استلام: 12 متر بسعر 120 ج
- **الجديد:** (1000 + 1440) / 22 = **110.91 ج/متر** ✅

تم اختبارها بـ 5 سيناريوهات مختلفة — كلها صح.

## 🔒 الأمان والتكامل

### Transactions ذرية
كل `saveReceipt` يعمل كل التغييرات داخل `upConfig` واحد:
- Receipt + Stock + Movements + Treasury + Checks + SupplierPayments

يعني: **يا كل حاجة تنجح، يا كل حاجة ترجع.**

### Validation متعدد المستويات
1. المورد مطلوب
2. على الأقل بند واحد بكمية > 0
3. إجمالي > 0
4. لو كاش: الخزنة + المبلغ مطلوبين
5. لو شيك: البنك + الرقم + المبلغ مطلوبين

### Backward Compatible
لو `stockEnabled = false`:
- الاستلام بيتسجل **بدون تحديث للمخزن**
- الدفع والكشف بيشتغلوا عادي
- مفيش stockMovements
